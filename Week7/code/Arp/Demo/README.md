# send-arp
Attack arp table of victim

## usage
```shell
syntax : send-arp <interface> <sender ip> <target ip> [<sender ip 2> <target ip 2> ...]
sample : send-arp wlan0 192.168.10.2 192.168.10.1
```

## description

* `sender ip` is the victim's ip.
* `target ip` is the gateway's ip.

* `attacker` is **you**.
